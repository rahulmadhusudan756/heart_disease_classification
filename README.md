# HeartGuard AI - Heart Disease Classification

AI-powered heart disease classification system using machine learning models.

## Quick Start

### Backend

```bash
cd /Users/anwerbasha/projects/heart_disease_classification
source .venv/bin/activate
uvicorn main:app --reload
```

Backend runs at: http://localhost:8000

### Frontend

```bash
cd frontend
npm run dev
```

Frontend runs at: http://localhost:3000

## Models

- **Normal Heart** - Initial cardiac screening
- **Heart Attack** - Myocardial infarction detection
- **Heart Failure** - Heart failure prediction
- **Hypertension** - Hypertension risk assessment
- **Coronary** - Coronary artery disease detection
